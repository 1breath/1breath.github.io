### 定义
Python推导式， 利用已有数据序列 构建 新的数据序列 结构体。

### 类型（四种：list, dict, set, tuple）

##### 列表（list）推导式
格式为：[表达式 for 变量 in 列表] 
              [out_exp_res for out_ex in input_list]
              或
              [表达式 for 变量 in 列表 if 条件] 
              [out_exp_res for out_ex in input_list if condition]
实例1：

  multiples=[i for i in range(30) if i%3==0]
  print(multiples)

 实例2:
 >>> names = ['Bob','Tom','alice','Jerry','Wendy','Smith']  
>>> new_names = [name.upper()for name in names if len(name)>3]  
>>> print(new_names)  
['ALICE', 'JERRY', 'WENDY', 'SMITH']

##### 字典（dict）推导式
格式：
{key_expr: value_expr for value in collection} 
或
{key_expr: value_expr for value in collection if conditional}

实例：使用字符串及其长度创建字典
listdemo=['Google','Runoob','Taobao']
newdict={key:len(key) for key in listdemo}
newdict
{'Google':6,'Runoob':6,'Taobao':6}

##### 集合（set）推导式
格式：
{expression for item in Sequence}
或
{expression for item in Sequence if conditional}
实例：
计算1,2,3的平方数
set_new={i**2 for i in (1,2,3)}

##### 元组（tuple）推导式
用途：可利用range区间、元组、列表、字典和集合等数据类型，快速生成一个满足指定需求的元组。元组推导式返回的结果是一个生成器对象？
格式：
(expression for item in Sequence)
或
(expression for item in Sequence if conditional)
实例：生成一个包含数字1-9的元组。
a=(x for x in rang(1,10))
a
tuple(a)
