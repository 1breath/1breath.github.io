# 编码
Python 3 源码文件默认以UTF-8编码。可以自行指定编码，编码不同对应的字符编码不同。

# 标识符
即创造的新词
首字符，须为字母或下划线_
标识符由字母、数字、下划线组成
区别大小写。及全角和半角字符（汉语）

# 保留字／关键字
系统中保留在keyword模块中、不可用作自建标识符名称的关键字。可用input keyword模块，
keyword.kwlist 方法，列出保留字
['False', 'None', 'True', 'and', 'as', 'assert', 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except', 'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda', 'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try', 'while', 'with', 'yield']

# 注释
三种方法
## 一、单行注释，以#开头
## 二、多行注释，每行以#开头
## 三、多行注释，以连续三个单引号‘’‘或三个双引号“”“，包括。

# 行与缩进
用缩进表示代码块。
缩进的空格数可变，但同一代码块须包含相同的缩进。

# 多行语句
长语句可以反斜杠\，实现，类似于文档处理中的回车键

# 数字类型
四种：整数、浮点数(小数点或者科学计数法)、布尔型（True、Flase）、复数（m+nj、1.3+2.5j)

# 字符串
以单引号或双引号表达
三引号可以指定多行字符串
转义符反斜杠\
用r，阻止反斜杠转义
字符串可用运算符+连接，用运算符*\重复
两种索引方式，自左至右，从0开始，自右至左，以-1开始。
字符串不能改变
单字符也是字符串，长度为1。
字符串截取语法：变量[头下标：尾下标：步长]

# 空行
不计入语法，分割不同功能与含义代码，便于维护、重构。

# 同一行显示多条语句
语句间用分号；分割

# 代码组
缩进相同的一组语句（子句）构成的一个代码块（句群、段落）
主要是if,while,def, class, elif, else等复合语句

# 输出 
print()函数
不需要换行输出时，可以变量末尾加end=""

# import 与from...import

导入相应模块
导入整个模块：import somemodule
导入某模块中的某个函数：from somemodule import somefuction
导入某模块中的多个函数：from somemodule import firstfuc, secondfuc, thirdfuc
导入某模块中的全部函数 from somemodule import *

# 命令行参数
格式：-字母，如-h,-c,-d
使用脚本形式执行Python时，可以接收命令行输入的参数
