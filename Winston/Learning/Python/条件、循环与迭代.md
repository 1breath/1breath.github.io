或
match...case：对if-else的简化
格式：
match subject:
    case <pattern_1>:
        <action_1>
    case <pattern_2>:
        <action_2>
    case <pattern_3>:
        <action_3>
    case _:
        <action_wildcard>
case _: 类似于 C 和 Java 中的 default:，当其他 case 都无法匹配时，匹配这条，保证永远会匹配成功。

## 循环
#### Python中的循环语句有两个：_for_ 和 _while_ 
Python循环语句的 控制结构 图如下所示：
![[Pasted image 20230217215621.png]]
也就是当条件为真时，执行代码块后，继续条件判断，进入下一轮循环。条件为假时，则退出循环。Control+C 组合键，退出无限循环。 

##### while 语句
![[Pasted image 20230217220142.png]]
while语句的一般形式为：
while 决断条件(condition)
     执行语句。

##### 无限循环
设置条件表达式为永远为真来实现。

##### while循环使用else语句
当while后条件判断语句为false时，执行else语句块
语法格式为：
```
	while <expr>:
		<statement(s)>
	else:
		<additional_statement(s)>
```

#### for语句
一般格式
```
for <variable> in <sequence>:
	<statement>
else:
	<statement>
```
